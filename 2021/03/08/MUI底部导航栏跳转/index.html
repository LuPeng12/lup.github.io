MUI底部导航栏跳转问题

2021-03-08 | 技术 | 阅读

导航页面————dh.php

<div  style="position: fixed;bottom: 0px;width: 100%;height: 40px;">
        <nav class="mui-bar mui-bar-tab"  >
            <a class="mui-tab-item <?php echo (B_get('dh')=='1'?'mui-active':''); ?>"   href="xxx.php?dh=1">
                <span class="mui-icon mui-icon-home"></span>
                <span class="mui-tab-label">页面1</span>
            </a>
            <a class="mui-tab-item  <?php echo (B_get('dh')=='2'?'mui-active':''); ?>"     href="xxx2.php?dh=2">
                <span class="mui-icon mui-icon-chatboxes"></span>
                <span class="mui-tab-label">页面2</span>
            </a>
            <a class="mui-tab-item  "   href="/wx/default.php">
                <span class="mui-icon mui-icon-gear"></span>
                <span class="mui-tab-label">返回</span>
            </a>
        </nav>
    </div>
<script type="text/javascript">
    mui('.mui-bar-tab').on('tap','a',function(){
        // console.log(this.getAttribute('href'));
        location.href = this.getAttribute('href');
    })
</script>

其他导航所至的页面只需要添加dh.php

<?php
    include_once ('dh.php');
    ?>
